adminItem(self, **kw)

source code 

Provee la lista de items de una fase en un proyecto Parametros: 'msg' (opcional) se utiliza para desplegar algun mensaje de error. 'faseid' es el id de la fase de la cual se estan solicitando la lista de sus items Retorna: lista de items (VersionItem) de la fase solicitada
